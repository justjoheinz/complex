This repository provides methods to calculate with complex numbers.

```moonbit
let a =  {real: 1.0, imag: -1.0}
// Addition
let b = a + a
// Substraction
let c = b - a
// Multiplication
let d = a * b 
// Negation
let e = - a
// Division
let f = a / b 
// Abs / length
let g = a.abs()
// Conjugate
let h = a.conjugate()
// Reciprocal
let i = a.reciprocal()
```

## Installation

``` sh
moon add justjoheinz/complex
```
